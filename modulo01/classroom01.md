# 📢 Descrição – Aula 01

Nesta aula iniciaremos nossa jornada no mundo do versionamento de código. Você aprenderá o que é versionamento, por que o Git foi criado e quais problemas ele resolve em projetos de tecnologia. Também conhecerá a diferença entre Git e GitHub, além da importância dessas ferramentas no mercado de trabalho.

### 📚 Conteúdos abordados

* O que é versionamento;
* Histórico de alterações;
* Rastreabilidade;
* Git x GitHub;
* A importância do Git no mercado de trabalho.

### 🛠️ Atividade prática

* Criar uma conta no GitHub;
* Explorar a plataforma;
* Registrar as anotações da aula;
* Realizar o Caderno de Atividades da Aula 01.

### 🎯 Objetivo

Compreender os fundamentos do versionamento e iniciar a preparação para a construção do projeto integrador **Meu Manual de Git e GitHub**.

### 🚀 Importante

Guarde seu nome de usuário e senha do GitHub, pois sua conta será utilizada durante toda a disciplina.
