# 🚀 DESAFIO DA AULA

## Situação-Problema

Uma empresa possui um sistema desenvolvido há vários anos.

O desenvolvedor responsável saiu da empresa e deixou apenas os arquivos do projeto.

Não existe:

* README;
* Documentação;
* Manual de instalação;
* Explicação sobre o funcionamento do sistema.

A nova equipe precisa continuar o desenvolvimento.

---

### Desafio

* Quais problemas essa equipe poderá enfrentar;
* Como a documentação poderia ajudar;
* Qual a importância do README para novos desenvolvedores.
