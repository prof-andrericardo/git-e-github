# 📢 Descrição – Aula 02

Nesta aula conheceremos o GitHub como uma plataforma utilizada por estudantes, desenvolvedores e empresas de tecnologia para armazenar projetos, compartilhar conhecimento e construir portfólios profissionais. Também iniciaremos oficialmente o projeto integrador **Meu Manual de Git e GitHub**.

### 📚 Conteúdos abordados

* O que é GitHub;
* O que é um repositório;
* Diferença entre Git e GitHub;
* GitHub no mercado de trabalho;
* GitHub como portfólio profissional;
* Estrutura inicial do Projeto Integrador;
* Introdução ao README.

### 🛠️ Atividade prática

* Analisar o próprio perfil GitHub;
* Criar a estrutura inicial do Projeto Integrador;
* Criar o arquivo README.md;
* Organizar as pastas das aulas;
* Produzir o resumo da Aula 02.

### 🎯 Objetivo

Compreender o papel do GitHub no mercado de tecnologia e iniciar a construção do projeto **Meu Manual de Git e GitHub**, que será desenvolvido ao longo de toda a disciplina.

### 🚀 Entregáveis da Aula

✔ Perfil GitHub revisado.

✔ Estrutura inicial do projeto criada.

✔ README.md criado.

✔ Pasta das aulas organizada.

✔ Resumo da Aula 02 produzido.

### 💡 Importante

O GitHub não é apenas um local para armazenar projetos. Ele também funciona como um portfólio profissional, permitindo demonstrar conhecimentos, organização e evolução ao longo do tempo.
