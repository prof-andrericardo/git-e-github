# 🎯 Princípio Pedagógico

O objetivo da disciplina não é formar especialistas em Git em 12 aulas.

O objetivo é permitir que o estudante:

* Entenda o propósito do versionamento;
* Desenvolva hábitos de documentação;
* Aprenda a colaborar;
* Utilize Git e GitHub com segurança em projetos futuros.

---

# 🏆 Resultado Esperado

Ao final de cada aula o professor deverá observar evidências de que os estudantes:

✅ Compreenderam os conceitos trabalhados.

✅ Conseguiram aplicá-los.

✅ Registraram a aprendizagem em seus repositórios.

✅ Evoluíram gradualmente no Projeto Integrador.

✅ Desenvolveram autonomia e organização.
