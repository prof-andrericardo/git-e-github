## 📚 Sobre a Disciplina

Bem-vindo ao repositório oficial da disciplina **Versionamento de Código com Git e GitHub**.

Nesta disciplina aprenderemos os fundamentos do controle de versão utilizando Git e GitHub, compreendendo desde os conceitos básicos até fluxos de trabalho utilizados por equipes profissionais de desenvolvimento de software.

Ao longo das aulas cada estudante desenvolverá seu próprio projeto:

> 🚀 Meu Manual de Git e GitHub

que servirá como material de estudo, documentação técnica e portfólio inicial.

---

## 🎯 Objetivos da Disciplina

Ao final do curso você deverá ser capaz de:

* Compreender o propósito do versionamento;
* Utilizar Git e GitHub;
* Produzir documentação utilizando Markdown;
* Registrar alterações através de commits;
* Trabalhar com branches;
* Resolver conflitos simples;
* Criar Pull Requests;
* Organizar projetos de forma profissional.
